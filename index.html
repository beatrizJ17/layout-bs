Módulo 1: PC Hardware Incluye las funciones de los componentes de un ordenador personal, el diagnóstico y reparación de problemas de hardware, y la seleccion y recomendación de hardware adecuado.

Módulo 2: Sistemas operativos Incluye la instalación y actualización de los sistemas operativos y aplicaciones más comunes y el uso de las herramieentas del sistema para la solución de problemas y reparación de los sistemas operativos.

Sistema informático. Un sistema informático se compone de componentes de hardware y de software.

Hardware y software. El hardware es el equipo fisico como es el caso, unidades de almacenamiento, teclados, monitores, cables, altavoces e impresoras. El software es la operación programas del sistema y el sistema operativo instruye a la computadora cómo operar, programs o aplicaciones realizar diferentes funciones.

La caja del ordenadorProporciona la protección y el apoyo a los componentes internos. De ser durable, facil de mantener y tener sufuciente espacio para la expanción, El tamaño y diseño de un caso recibe el nombre de FACTOR DE FORMA.

Fuente de alimentación Convierte alimentación de CA de la toma de corriente en DC. Debe proveer suficiente energía para los componentes instalados y adiciones futuras.
